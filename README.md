# Multi-Firm Growth Command Center

**A strategic prototype demonstrating centralized organic growth operations for PE-backed CPA & advisory platforms.**

Built by [Binil Chacko](https://linkedin.com/in/binilchacko) as a Chief Growth Officer strategic demonstration.

## What This Tool Does

This interactive command center shows how a CGO would operationalize organic growth across a multi-firm professional services platform:

- **Platform Overview** — Strategic diagnosis of growth gaps with a 90-day sprint plan
- **Firm Intelligence** — Individual firm growth readiness scoring and recommendations
- **Cross-Sell Matrix** — Service-to-service conversion pathways and revenue potential
- **Pipeline Health** — Full funnel analytics from lead generation to close
- **Service Expansion** — Gap analysis showing highest-ROI growth targets
- **ICP Scorer** — Interactive prospect qualification tool for firm partners
- **Growth Model** — Revenue projection modeling CGO impact vs baseline

## Tech Stack

- React 19 + Vite
- Recharts for data visualization
- Deployed on Vercel

## Local Development

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and connect to Vercel — it auto-detects the Vite framework.
